# 项目总览

当前项目主要包含两个独立作业目录：

## `hw3`

`hw3` 完成的是原油期货期限结构分析：

- 获取上海国际能源交易中心原油 `SC` 不同月份合约的最近收盘价
- 绘制期限结构图
- 判断市场是升水 `Contango` 还是贴水 `Backwardation`
- 结合期限结构给出可能的跨期交易思路
- 生成对应的 PDF 报告

## `pj1`

`pj1` 完成的是一个较完整的量化交易系统回测：

- 基于原油主力连续合约 `SC0` 构建双均线趋势策略
- 包含开仓、平仓、资金管理和风控模块
- 使用真实历史数据进行回测
- 输出胜率、赔率、最大回撤、收益率、夏普比率等指标
- 生成策略图表和 PDF 报告

## 目录说明

```text
hw3/   原油期限结构分析作业
pj1/   原油双均线交易系统作业
```
