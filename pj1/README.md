# PJ1 原油期货双均线交易系统

## 项目目标

本项目完成一个较完整的量化交易系统，基于上海国际能源交易中心原油主力连续合约 `SC0` 构建双均线趋势策略，并补齐以下部分：

1. 开仓规则
2. 平仓规则
3. 资金管理
4. 风险控制
5. 回测指标评估

## 策略设计

### 1. 交易标的

- 标的：上海国际能源交易中心原油主力连续合约 `SC0`
- 数据来源：`akshare`
- 回测区间：`2019-01-02` 到 `2026-03-20`

### 2. 开仓规则

- 当 `MA10 > MA40` 时，建立多头仓位。
- 当 `MA10 < MA40` 时，建立空头仓位。

这里采用趋势跟踪思路：

- 短期均线上穿长期均线，说明价格进入上升趋势，系统做多。
- 短期均线下穿长期均线，说明价格进入下降趋势，系统做空。

### 3. 平仓规则

系统存在三类平仓机制：

1. 反向信号平仓：均线方向发生反转时平仓或直接反手。
2. 硬止损平仓：价格触发 `2 x ATR` 的止损距离时离场。
3. 跟踪止损平仓：盈利后若价格从阶段最优位置回撤超过 `3 x ATR`，则离场锁定利润。

### 4. 资金管理

- 初始资金：`1,000,000`
- 单笔风险：账户权益的 `2%`
- 最大仓位：账户权益的 `80%`

仓位计算逻辑：

- 先根据 ATR 估算单笔止损幅度。
- 再用 `风险预算 / 止损幅度` 得到目标仓位比例。
- 如果目标仓位过大，则限制在 `80%` 以内。

### 5. 风险控制

- 手续费：`0.03%`
- 滑点：`0.02%`
- ATR 硬止损
- ATR 跟踪止损
- 单边仓位上限 `80%`

## 回测结果

本次回测使用 `pj1_backtest.py` 实际运行得到的指标如下：

| 指标 | 数值 |
|------|------|
| 期末权益 | `1,543,829.70` |
| 累计收益率 | `54.38%` |
| 年化收益率 | `6.46%` |
| 最大回撤 | `-23.03%` |
| 夏普比率 | `0.54` |
| 交易次数 | `93` |
| 胜率 | `38.71%` |
| 赔率 | `2.04` |
| 平均单笔收益率 | `0.29%` |
| Profit Factor | `1.29` |

结果解读：

- 胜率不高，但赔率大于 `2`，说明策略依靠“赚大亏小”获利。
- 最大回撤约 `23%`，在趋势跟踪类策略中属于可以接受但仍需关注的水平。
- 夏普比率为 `0.54`，说明策略有一定收益风险比，但还存在优化空间。

## 结论

该项目已经形成一套较完整的交易系统：

1. 有明确的开仓和平仓规则。
2. 有基于风险预算的仓位管理。
3. 有手续费、滑点、止损和跟踪止损等风控模块。
4. 有完整的量化评价指标。

从本次回测结果看，这套双均线策略在原油主力连续合约上是可以盈利的，但收益率和夏普比率仍然有进一步优化空间，后续可以继续尝试：

- 优化均线参数
- 引入波动率过滤
- 引入突破确认或成交量过滤
- 针对不同市场阶段做参数分层
